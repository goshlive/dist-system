
### 1. Basic Socket Connection (`connect/`)
A minimal example demonstrating fundamental socket communication between client and server.

**What it demonstrates:**
- Establishing TCP socket connections
- Basic client-server communication pattern
- Simple message sending and receiving

**Key concepts:**
- Socket creation and binding
- Connection handling
- Basic I/O operations

### 2. Interactive Communication (`interactive/`)
A chat-like application demonstrating bidirectional, interactive socket communication.

**What it demonstrates:**
- Continuous two-way communication
- Multiple message exchanges
- Real-time interaction patterns

**Key concepts:**
- Persistent connections
- Bidirectional data flow
- Interactive session handling
